## 방어구 세트 구조

### 세트 정보
- **세트 이름**: "어느 말괄량이의 탐사복"
- **부위**: 상의, 하의, 어깨, 벨트, 신발 (총 5개)
- **접두어**: 전격, 허상

### ARMOR_DISPLAY_NAMES 구조 (shared_data.js)
```javascript
"어느 말괄량이의 탐사복": {
    "상의": ["못말리는 말괄량이의 가죽 자켓", "어느 말괄량이의 가죽 자켓"],
    "하의": "어느 말괄량이의 버블 반바지",
    "어깨": "어느 말괄량이의 특수 고글",
    "벨트": "어느 말괄량이의 벨트",
    "신발": "어느 말괄량이의 신발"
}
```

## 장비 등급 및 타입

### 1. 기본 장비 (접두어 없음)
- 어느 말괄량이의 가죽 자켓 (상의 - 일반)
- 어느 말괄량이의 버블 반바지 (하의)
- 어느 말괄량이의 특수 고글 (어깨)
- 어느 말괄량이의 벨트 (벨트)
- 어느 말괄량이의 신발 (신발)

### 2. 접두어 장비
**전격 접두어 (5개):**
- 전격: 어느 말괄량이의 가죽 자켓
- 전격: 어느 말괄량이의 버블 반바지
- 전격: 어느 말괄량이의 특수 고글
- 전격: 어느 말괄량이의 벨트
- 전격: 어느 말괄량이의 신발

**허상 접두어 (5개):**
- 허상: 어느 말괄량이의 가죽 자켓
- 허상: 어느 말괄량이의 버블 반바지
- 허상: 어느 말괄량이의 특수 고글
- 허상: 어느 말괄량이의 벨트
- 허상: 어느 말괄량이의 신발

### 3. 익시드 장비 (상의만 존재)
- **장비명**: "못말리는 말괄량이의 가죽 자켓" (ARMOR_DISPLAY_NAMES의 상의 첫 번째)
- **특징**: 무조건 접두어 + 익시드 태그가 붙음
- **익시드 태그**: [이상], [선봉], [의지]
- **접두어**: 전격, 허상

**익시드 장비 6개:**
1. [이상] 전격: 못말리는 말괄량이의 가죽 자켓
2. [선봉] 전격: 못말리는 말괄량이의 가죽 자켓
3. [의지] 전격: 못말리는 말괄량이의 가죽 자켓
4. [이상] 허상: 못말리는 말괄량이의 가죽 자켓
5. [선봉] 허상: 못말리는 말괄량이의 가죽 자켓
6. [의지] 허상: 못말리는 말괄량이의 가죽 자켓

## 세트 효과 판정 규칙

### 기본 원칙
1. **접두어 세트** 먼저 확인
2. 접두어 세트가 없으면 **기본 세트** 확인
3. **익시드 태그는 무시**, 접두어만 세트 판정에 사용

### 세트 카운트 방법
| 장비 형태 | 세트 카운트 |
|---------|----------|
| `[이상] 전격: 못말리는 말괄량이의 가죽 자켓` | **전격** 카운트 |
| `전격: 어느 말괄량이의 버블 반바지` | **전격** 카운트 |
| `허상: 어느 말괄량이의 특수 고글` | **허상** 카운트 |
| `어느 말괄량이의 신발` | **기본** 카운트 |

### 세트 효과 발동 조건
- **3세트**: 같은 접두어 3개 이상
- **5세트**: 같은 접두어 5개

### 세트 효과 데이터 위치
`armor_set_effects.js` 파일 참조

**예시 (전격 3세트):**
```javascript
"전격": {
    "3": {
        "label": "공격속도 +10%\n캐스팅속도 +15%\n이동속도 +10%\n수공격속성\n명공격속성\n감전 데미지 10% 증가\n모든 타입 피해 20% 증가\n공격 시 4% 감전 데미지",
        "stats": [
            { "stats": ["공격속도"],            "amount": 10, "unit": "%" },
            { "stats": ["캐스팅속도"],          "amount": 15, "unit": "%" },
            { "stats": ["이동속도"],            "amount": 10, "unit": "%" },
            { "stats": ["감전 데미지 증가"],    "amount": 10, "unit": "%" },
            { "stats": ["모든 타입 피해 증가"], "amount": 20, "unit": "%" },
            { "stats": ["공격 시 감전 데미지"], "amount": 4,  "unit": "%" }
        ]
    }
}
```

## 예시

### 예시 1: 전격 3세트 발동
**보유 장비:**
1. 전격: 어느 말괄량이의 버블 반바지 (하의)
2. 전격: 어느 말괄량이의 특수 고글 (어깨)
3. 전격: 어느 말괄량이의 벨트 (벨트)
4. 어느 말괄량이의 신발 (신발)
5. 어느 말괄량이의 가죽 자켓 (상의)

**세트 판정:**
- **전격**: 3개 (하의 + 어깨 + 벨트) → ✅ **전격 3세트 효과 발동**
- **기본**: 2개 (신발 + 상의) → 세트 미달성

**적용 효과:**
```
공격속도 +10%
캐스팅속도 +15%
이동속도 +10%
감전 데미지 증가 +10%
모든 타입 피해 증가 +20%
공격 시 감전 데미지 +4%
```

**적용 효과:**
```

감전 데미지 증가 +10%
모든 타입 피해 증가 +20%

```

## 중요 사항

### 제약사항
- ✅ **비교 모드에서만** 사용하는 로직
- ✅ **다른 곳의 로직은 건들지 않음**

### 세트 효과 중복 적용
- 3세트와 5세트 효과는 **모두 적용**됨 (누적)
- 예: 전격 5세트 = 전격 3세트 효과 + 전격 5세트 효과

### 우선순위
1. 접두어 세트 먼저 확인 (전격, 허상 등)
2. 접두어 세트가 없으면 기본 세트 확인
3. 둘 다 없으면 세트 효과 없음

## 데이터 참조 위치

### shared_data.js
```javascript
ARMOR_SETS          // 세트별 부위 정보
ARMOR_PREFIX        // 세트별 접두어 정보
ARMOR_DISPLAY_NAMES // 세트별 실제 아이템명
```

### armor_set_effects.js
```javascript
ARMOR_SET_EFFECTS   // 세트 효과 데이터
// 구조: ARMOR_SET_EFFECTS[아이템명][접두어]["3" or "5"]
```
