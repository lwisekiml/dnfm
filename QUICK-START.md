# ⚡ 빠른 시작 가이드

> **5분 안에 RPG 장비 관리 시스템 시작하기**

---

## 🎯 목표

이 가이드를 따라하면:
- ✅ 첫 캐릭터 생성
- ✅ 장비 설정
- ✅ 스킬룬 설정
- ✅ 데이터 저장/복구
- ✅ 2개 캐릭터 비교

**소요 시간: 5분**

---

## 📋 Step 1: 서버 실행 (1분)

### Windows

```bash
# 명령 프롬프트 또는 PowerShell
cd 프로젝트폴더
python -m http.server 8000
```

### Mac/Linux

```bash
# 터미널
cd 프로젝트폴더
python3 -m http.server 8000
```

### 확인

```
Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/) ...
```

브라우저에서 열기: **http://localhost:8000**

---

## 🎮 Step 2: 첫 캐릭터 생성 (1분)

### 1. 기본 정보 입력

1. **"➕ 캐릭터 추가"** 버튼 클릭
2. 왼쪽 정보 칸에 입력:
   ```
   스탯: 힘 또는 지능 선택
   속강: 화속강/수속강/명속강/암속강 선택
   직업: 귀검사
   이름: 메인캐릭터
   항마력: 55000
   ```

### 2. 무기 설정

| 항목 | 입력값 |
|------|--------|
| 희귀도 | 에픽 |
| 익시드 | 이상 |
| 접두어 | (선택) |
| 아이템이름 | 내 무기 이름 |
| 강화 | +12 |

### 3. 마법봉인 설정

| 봉인 | 옵션 | 수치 |
|------|------|------|
| 고유 옵션 | 데미지 증가 | 8 |
| 일반 옵션 | 힘 또는 지능 | 46 |

**💡 팁:** 
- 속강 선택하면 자동으로 노란색 하이라이트!
- 스탯 선택하면 봉인 옵션도 노란색!

---

## ⚙️ Step 3: 방어구 설정 (1분)

### 빠른 설정

```
상의: 에픽, 강화 +10
하의: 에픽, 강화 +10
어깨: 에픽, 강화 +10
신발: 에픽, 강화 +10
벨트: 에픽, 강화 +10
```

### 세트 효과 확인

**같은 세트 3개 이상 착용 시 아이템 이름이 청록색으로 변해요!**

예시:
```
상의: 엠프리스 스파이더 상의
하의: 퀸 스파이더 하의
어깨: 퀸 스파이더 머리
→ 3개 착용하면 모두 청록색! ✨
```

---

## 💎 Step 4: 스킬룬 설정 (1분)

### 1. 모달 열기

**"스킬룬"** 행의 **"수정"** 버튼 클릭

### 2. 일괄 설정 (빠름!)

```
1. 상단 드롭다운에서 "각성" 선택
2. "룬 설정" 버튼 클릭
   → 20개 슬롯 모두 "각성"으로 설정됨!

3. 레벨 드롭다운에서 "IV" 선택
4. "레벨 설정" 버튼 클릭
   → 모든 룬 레벨 IV!

5. 스킬 레벨 "60" 선택
6. "레벨로 설정" 버튼 클릭
   → 모든 스킬 레벨 60!
```

### 3. 각인 입력 (선택)

```
각인 1: 공격력 50% 증가
각인 2: 크리티컬 30% 증가
```

### 4. 저장

**"저장"** 버튼 클릭 → 스킬룬 요약이 표시됨!

---

## 💾 Step 5: 데이터 저장 (30초)

### 자동 저장

**아무것도 안 해도 돼요!**
- 입력하면 자동으로 800ms 후 저장
- 우측 상단에 "💾 저장됨" 메시지 표시

### 수동 백업

```
1. "📤 JSON 파일 저장" 버튼 클릭
2. 다운로드 폴더에 자동 저장
   파일명 예시: dnfm_character_equipment_backup_2026-02-07_14-30.json
```

---

## ⚖️ Step 6: 비교 모드 (1분)

### 1. 두 번째 캐릭터 생성

```
"➕ 캐릭터 추가" 버튼 클릭
이름: 부캐릭터
무기 강화: +10 (메인은 +12였음)
```

### 2. 비교 모드 진입

1. 상단 **"⚖️ 비교"** 버튼 클릭
2. 왼쪽에서 "메인캐릭터" 선택
3. 오른쪽에서 "부캐릭터" 선택

### 3. 결과 확인

**4가지 비교 테이블 자동 생성!**

```
*장비 비교*
슬롯      강화    차이    강화
무기      +12   ⬆ +2    +10   ← 차이값 표시!
```

**색상 의미:**
- 🟢 **녹색 (⬆)** - 왼쪽이 더 높음
- 🔴 **빨간색 (⬇)** - 왼쪽이 더 낮음
- ⚫ **회색 (-)** - 동일

## ⚠️ 자주 묻는 질문

### Q1: 저장이 안 돼요!
**A:** 
```
1. 개발자 도구 열기 (F12)
2. Console 탭
3. localStorage.getItem('dnfm_character_equipment_data')
4. null이면 저장 안 된 거예요
5. 입력 후 1초 정도 기다려보세요!
```

### Q2: 세트 효과가 안 나와요!
**A:**
```
1. 희귀도가 "에픽"인지 확인
2. 같은 세트 이름 3개 이상인지 확인
3. 아이템 이름을 select에서 선택했는지 확인
   (직접 입력하면 인식 안 됨!)
```

### Q3: 비교 모드에서 선택이 안 돼요!
**A:**
```
1. 캐릭터가 2개 이상인지 확인
2. 직업/이름을 입력했는지 확인
3. 자동 저장 대기 (1초)
4. 비교 모드 다시 진입
```
